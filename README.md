# Forza Tkinter Toolkit

A powerful Python Tkinter app by **Forza** (GitHub: [Forzatech](https://github.com/Forzatech)) featuring:

- User registration & editing
- Financial calculations
- Built-in calculator
- Contact manager
- Base salary updates
- Authentication & secure logout

## Installation

```bash
git clone https://github.com/Forzatech/forza-tk-tools.git
